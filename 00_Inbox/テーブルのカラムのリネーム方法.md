# テーブルのカラムのリネーム方法

## 💡 学んだこと・解決した課題
- テーブルのカラムをリネームする方法
## 📝 詳細・原因
- すでに存在するカラムのリネーム方法
- STEPを戻すのではなく、新たにマイグレーションファイルを作成する
- マイグレーション後はモデル、ビュー、コントローラー、JS、以前のカラム名で定義している部分を新たなカラム名に変更する
## 💻 解決策・コード
- マイグレーションファイルを作成
```bash
docker compose exec web bin/rails g migration RenameNameToTitleInTtips
```
- 中身を追記
```ruby
class RenameNameToTitleInTtips < ActiveRecord::Migration[7.2]
	def change
		rename_column :trips, :name, :title
	end
end
```
- db:migrate
```bash
docker compose exec web bin/rails db:migrate
```
