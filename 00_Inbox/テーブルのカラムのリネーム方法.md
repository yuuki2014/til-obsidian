# テーブルのカラムのリネーム方法

## 💡 学んだこと・解決した課題
- テーブルのカラムをリネームする方法
## 📝 詳細・原因
- すでに存在するカラムのリネーム方法
- STEPを戻すのではなく、新たにマイグレーションファイルを作成する
- マイグレーション後はモデル、ビュー、コントローラー、JS、以前のカラム名で定義している部分を新たなカラム名に変更する
## 💻 解決策・コード
```bash
docker compose exec web bin/rails g migration 
```
